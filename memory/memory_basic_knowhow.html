Memory Basic Know-how

Memory Types

RAM

  • Random access memory, Main Memory OS will operate on

  • RAM is commonly implemented in a semiconductor tech called DRAM (Dynamic Random Access Memory)

ROM, also known as firmware

Read only memory, normally used to store static programs, like Bootstrap Program, which is used to start computer

EEPROM

  • Can be changed, but not frequently, contains mostly static programs

  • Smart phones use EEPROM to store their factory-installed programs

Secondary storage

Able to hold large quantity of data permanently, most common one is Magnetic Disk

CPU

  • CPU can only directly work with registers

  • CPU can load data from memory into register and also store data in register back to memory

  • In a Von-Neumann architecture's typical execution cycle, an instruction is first fetched from memory and stored in Instruction Register, the instruction is then decoded and may cause operands to be fetched from memory and stored in some other internal register. After the instruction is executed, result maybe stored back into memory.
